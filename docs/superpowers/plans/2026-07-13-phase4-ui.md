# Phase 4: UI Upgrades — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Shinies + `pet dex` collection, richer HUD (speech-bubble caption, streak flame, HP bar), battle FX (type-colored particles, done-flash + shake), and a proper evolution cinematic — on the overlay, with terminal/statusline adaptations.

**Architecture:** All game state stays in the core: shiny is rolled at partner-write (pack-configured 1/64, pokemon only), `hp_pct` is computed into `resolved.json` from the existing counters, dex gains shiny upgrades and a `dex` subcommand. Renderers only draw new fields. Overlay FX are guarded enhancements — any FX failure must never break the pet (try/catch, no game impact).

**Tech Stack:** existing bash+jq core, JXA/QuartzCore (CAEmitterLayer), Python renderer, PokeAPI shiny gen-5 animated set (URL pattern verified live: `…/black-white/animated/shiny/<id>.gif`).

## Global Constraints

- All Phase 1–3 invariants (hook silence, bash 3.2, stdlib Python, locks for cache mutations — shiny roll happens inside `write_partner`'s existing single-shot write, no new lock needed).
- Digimon never rolls shiny (`shiny_rate` only in the pokemon pack); `hp_pct = clamp(100 - 15*mistakes + 10*tasks, 10, 100)` — core-computed, both franchises.
- Test seam: `PET_SHINY=0|1` forces the roll (env, like PET_SEED).
- Overlay layout shifts to fit the HP bar: exp bar y=26, hp bar y=21, mood label y=4 (pill unchanged).
- Version 0.7.0 at the end.

`write_partner` — replace the jq call:

```bash
write_partner() { # <pack-file> <line-index>
    local tmp rate s
    tmp="$(mktemp)"
    rate="$(jq -r '.sprites.shiny_rate // 0' "$1")"
    s=false
    if [ "$rate" -gt 0 ] 2>/dev/null; then
        [ $(( RANDOM % rate )) -eq 0 ] && s=true
        case "${PET_SHINY:-}" in 1) s=true ;; 0) s=false ;; esac   # test seam;
        # inside the rate guard so it can never shiny a franchise that has none
    fi
    jq --argjson i "$2" --arg d "$TODAY" --argjson sd "$RANDOM" --argjson sh "$s" \
       '{franchise: .franchise, line: .lines[$i].mons, type: .lines[$i].type,
         date: $d, seed: $sd, shiny: $sh}' \
       "$1" > "$tmp" && mv "$tmp" "$CACHE/partner"
    cmd_resolve
    echo "pet: $(jq -r '.line | join(" → ")' "$CACHE/partner")"
}
```

`update_dex` — shiny-aware:

```bash
update_dex() {
    [ -f "$CACHE/dex.json" ] || echo '[]' > "$CACHE/dex.json"
    local sp fr sh tmp
    sp="$(jq -r '.species' "$CACHE/resolved.json")"
    fr="$(jq -r '.franchise' "$CACHE/resolved.json")"
    sh="$(jq -r '.shiny' "$CACHE/resolved.json")"
    tmp="$(mktemp)"
    jq --arg s "$sp" --arg f "$fr" --arg d "$TODAY" --argjson sh "$sh" \
       'if any(.[]; .species == $s and .franchise == $f)
        then map(if .species == $s and .franchise == $f and $sh then .shiny = true else . end)
        else . + [{species: $s, franchise: $f, date: $d, shiny: $sh}] end' \
       "$CACHE/dex.json" > "$tmp" && mv "$tmp" "$CACHE/dex.json"
}
```

`cmd_dex` + dispatcher entry `dex) cmd_dex ;;`:

```bash
cmd_dex() {
    [ -f "$CACHE/dex.json" ] || echo '[]' > "$CACHE/dex.json"
    local f pack total caught
    for f in pokemon digimon; do
        pack="$(pack_file "$f")"; [ -f "$pack" ] || continue
        total="$(jq '.species | length' "$pack")"
        caught="$(jq --arg f "$f" '[.[] | select(.franchise == $f)] | length' "$CACHE/dex.json")"
        echo "$f: caught $caught/$total"
    done
    echo "shiny: $(jq '[.[] | select(.shiny)] | length' "$CACHE/dex.json") ✨"
    jq -r 'sort_by(.date)[] | "  \(.date)  \(.species)\(if .shiny then " ✨" else "" end)"' "$CACHE/dex.json"
}
```

```javascript
  var TYPE_RGB = { fire: [1,.5,.2], water: [.3,.6,1], grass: [.4,.9,.4], electric: [1,.9,.3],
    psychic: [1,.4,.8], normal: [.9,.9,.8], fighting: [.8,.4,.3], rock: [.7,.6,.4],
    ground: [.8,.7,.4], poison: [.7,.4,.9], bug: [.7,.9,.3], flying: [.7,.8,1],
    ghost: [.5,.4,.9], ice: [.6,.9,1], dragon: [.5,.5,1], vpet: [.75,.95,.6] };
  var emitter = null, fxColor = '';
  function setupEmitter(type) {
    try {
      var rgb = TYPE_RGB[type] || TYPE_RGB.normal;
      if (emitter) { emitter.removeFromSuperlayer; emitter = null; }
      var img = $.NSImage.alloc.initWithSize($.NSMakeSize(8, 8));
      img.lockFocus;
      $.NSColor.colorWithSRGBRedGreenBlueAlpha(rgb[0], rgb[1], rgb[2], 1).set;
      $.NSBezierPath.bezierPathWithOvalInRect($.NSMakeRect(0, 0, 8, 8)).fill;
      img.unlockFocus;
      var cell = $.CAEmitterCell.emitterCell;
      cell.setContents(img.CGImageForProposedRectContextHints(null, null, null));
      cell.setBirthRate(1); cell.setLifetime(0.6);
      cell.setVelocity(90); cell.setVelocityRange(50);
      cell.setEmissionRange(Math.PI * 2);
      cell.setScale(0.7); cell.setScaleRange(0.3); cell.setAlphaSpeed(-1.8);
      emitter = $.CAEmitterLayer.layer;
      emitter.setEmitterPosition($.CGPointMake(winW / 2, 140));
      emitter.setBirthRate(0);
      emitter.setEmitterCells($([cell]));
      win.contentView.layer.addSublayer(emitter);
      fxColor = type;
    } catch (e) { emitter = null; }
  }
  var fxOffUntil = 0, shakeUntil = 0;
  function burst(rate, ms) {
    try { if (emitter) { emitter.setBirthRate(rate); fxOffUntil = Date.now() + ms; } } catch (e) {}
  }
```

In `move()`/tick: if `Date.now() > fxOffUntil && emitter` → `emitter.setBirthRate(0)`; while `Date.now() < shakeUntil` add `(Math.random()*6-3)` to both position axes.
